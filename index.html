Exo 1

étapes en ligne de commandes

création du dossier : mkdir testOne

ouvrir ce dossier : cd testOne

création du fichier vide index.html : type nul > index.html

création du repo github : New repo

creation du fichier readme : echo "# AfciTest1" >> README.md

initialisation de git : git init

creation du fichier readme : git add README.md

sauvegarde du fichier : git commit -m "initialisation commit"

choix de la branch Main : git branch -M main

ajout du chemin vers github : git remote add origin https://github.com/Diverflo/AfciTest1.git

envoie du fichier readme sur github : git push -u origin main

prise en compte de la page index : git add index.html

message d'envoie : git commit -m "push index.html"

transfert de l'index.html : git push -u origin main

clonage du projet sur l'ordi pour le récupérer : git clone https://github.com/Diverflo/AfciTest1.git

se positionner dans le dossier : cd AfciTest1

sauvegarder tout : git add .

message d'envoie : git commit -m "push all"

envoyer tout : git push -u origin main
